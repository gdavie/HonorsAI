Detailed Description

This class is to contain the set of register collections so we can refer to them later.

Each instance from the training set will contain a set of registers for each program.

Diagnostics and logging have been stripped out - the rewrite broke them all anyways. The dimensionality of the arrays has been hidden with arraylist-like calls

Author
scobleaaro 070712 Stripped out version, modified to run on arrays rather than nested arraylists.
