Detailed Description

Used to store the final register values for an LGP program for EVERY training example. This is important to do if you want any form of cacheing whatsover.

DEPRECATED TO SOME DEGREE. this was intended to be done for every subprogram of a program, but this proved too costly in terms of memory and probably wouldn't be memory efficient anyway.

REPLACED with caching OLD PROGRAMS.
