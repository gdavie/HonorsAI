Detailed Description

This class represents some collection of registers. If they are read-only just make the collection const.

Member Function Documentation

int core.RegisterCollection.largestRegisterIndex ( )

Returns the register index of the register with the highest value (used in, e.g. winner takes all). If two registers are equal it returns the one with the lower index. At the moment i see no way to do these here, must have concrete types.
